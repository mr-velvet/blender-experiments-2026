# Sources

Assets externos baixados pra inspecao/comparacao. Nao versionados (ver `.gitignore`).

Pra reproduzir:

```bash
mkdir -p rigging/assets
curl -L -o rigging/assets/CesiumMan.glb \
  https://raw.githubusercontent.com/KhronosGroup/glTF-Sample-Assets/main/Models/CesiumMan/glTF-Binary/CesiumMan.glb
curl -L -o rigging/assets/Fox.glb \
  https://raw.githubusercontent.com/KhronosGroup/glTF-Sample-Assets/main/Models/Fox/glTF-Binary/Fox.glb
curl -L -o rigging/assets/RiggedFigure.glb \
  https://raw.githubusercontent.com/KhronosGroup/glTF-Sample-Assets/main/Models/RiggedFigure/glTF-Binary/RiggedFigure.glb
curl -L -o rigging/assets/BrainStem.glb \
  https://raw.githubusercontent.com/KhronosGroup/glTF-Sample-Assets/main/Models/BrainStem/glTF-Binary/BrainStem.glb
```
