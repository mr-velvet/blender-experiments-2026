# blender-experiments-2026

Experimentos de automacao do Blender via Python headless e MCP (Model Context Protocol).

## Setup
- Blender 5.1.2
- blender-mcp addon instalado (setup/addon.py)
- MCP server configurado no Claude Code (escopo user)

## Scripts
- `demo_cardboard_cube.py` — primeira demo: cubo + material Cardboard Outer + GLB
- `pipeline/` — pipeline de baking + batch generation (em construcao)
- `viewer/` — pagina HTML com model-viewer pra inspecionar GLBs
